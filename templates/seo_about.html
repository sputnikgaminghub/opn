Our story

A simple, template timeline your team can customize.

Before the Opinion (OPN) launch

The team saw a repeated failure mode across campaigns: opaque rules, un-auditable distributions, and poor user safety. OPN started as an internal set of tools to standardize eligibility and reduce disputes.

Prototype phase

We built the first public dashboards, added referral attribution, and implemented basic integrity checks for common abuse patterns.

Today

OPN focuses on making token operations easier to understand and harder to exploit, with a clean “Learn” layer for documentation and a production app for participation and accounting.
