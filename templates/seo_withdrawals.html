When withdrawals open

The platform can enforce an “open at” date for withdrawals. If withdrawals are not open yet, the app will show the exact opening date.

Default configuration targets 20th February 2026, but admins can adjust this before production.

Unlock requirement

  • Withdrawals unlock at 7 referrals.
  • The Withdrawals tab shows a progress bar (0/7 → 7/7).
  • Leaderboard access is available from the same area to help users track top referrers.

Fee model (anti-spam / validation)

The platform supports a small network fee that is validated per-chain (Ethereum / BSC). Default values are:

  • 0.00010 ETH on Ethereum
  • 0.0010 BNB on BSC

The app shows the active configuration before you submit a withdrawal request.

How the withdrawal flow works

  1. Open the airdrop dashboard and go to the Withdrawals tab.
  2. Check your status (open date, referral progress, and eligibility).
  3. When eligible, submit your withdrawal request. The UI indicates tokens are processed manually after submission.

Important anti-abuse note

The airdrop page warns that creating multiple accounts on the same device may result in a ban, and that only one account per device is eligible to withdraw.
